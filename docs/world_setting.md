# 道·衍世界設定 - 青雲門篇

## 1. 時代背景

**末法時代**

修仙文明已度過鼎盛期，靈氣衰竭，天地間的「仙氣」逐年遞減。三千年前的大能已成傳說，現今修士的上限是「度劫期」（Tier 5.5）。

### 主要事件線

- **昆侖秘戰**（300 年前）
  - 正道聯盟 vs 魔門的最後大戰
  - 青雲門掌門玄靈子身受重傷，修為停滯
  - 魔門勢力被壓制但未絕滅
  - 傳聞中有「終極秘寶」在昆侖山深處

- **靈氣衰變**（近 50 年）
  - 新生修士的修為上限從 Tier 4.0 降至 Tier 3.5
  - 古老靈脈逐漸失活
  - 大型宗門開始尋求外源補充（如「心悟仙草」）

## 3. 修為體系

### 九大境界

| 境界 | Tier 範圍 | 特徵 | 壽命 | 能力 |
|------|---------|------|------|------|
| 練氣期 | 1.0-1.9 | 初級修士，能感知靈氣 | +50 年 | 簡單法術 |
| 築基期 | 2.0-2.9 | 體內凝聚「靈基」 | +100 年 | 中級法術、劍法 |
| 金丹期 | 3.0-3.9 | 煉成「金丹」，戰力大增 | +300 年 | 跨域傳送、核心法術 |
| 元嬰期 | 4.0-4.9 | 凝聚「元嬰」，可分身 | +500 年 | 分身術、大規模攻擊 |
| 化神期 | 5.0-5.4 | 神念出竅，接近仙人 | +1000 年 | 半仙級法術 |
| 度劫期 | 5.5+ | 渡過天劫，證道成仙 | 不死（需度劫） | 仙級力量 |

**目前狀況**：
- 玄靈子是全大陸倖存的唯一度劫期修士
- 大多數長老停留在化神期或元嬰期
- 新弟子從練氣期 1.0 開始

### 修為進度機制

```
修為 = Tier 值（1.0-5.5）
進度 = cultivation_progress（0-100）

- Tier 1.0 → 1.9: 通過打坐、戰鬥、奇遇等積累進度
- 進度滿 100 時進階下一 Tier
- 突破時需要「靈感」或「契機」
- 跨越大境界（如 1.9→2.0）需要特殊事件或秘法
```

## 4. 戰鬥系統

### 勝率計算

```
基礎勝率 = 50%

同 Tier 內：
  每 0.1 差距 ±3% 勝率

跨 Tier（大境界）：
  每級差距 × 大境界懲罰
  - 低 0.5-1.4 級：-15% ~ -45% 勝率
  - 低 1.5-2.4 級：-65% ~ -85% 勝率
  - 低 2.5+ 級：< 5% 勝率（基本不可能）
```

### 傷害計算

```
基礎傷害 = 10 + Tier * 5 + (隨機 -5 到 +5)

修飾符：
- 使用特殊技能: × 1.5-2.0
- 克制對手: × 1.2-1.5
- 受傷狀態: × 0.7-0.9
- 法力不足: × 0.5（大招無法施放）
```

### 戰鬥風險

- **受傷**：HP 掉落
- **法力耗竭**：無法施法，需要休息恢復
- **失血過多**：HP < 0 時昏迷（可被救援）
- **重傷**：特定傷害後需要醫療

## 5. 修為突破機制

### 理論上升途徑

1. **打坐修煉**
   - 每日打坐 1 小時 = +1 進度
   - 靈氣充足地點加成：× 1.5

2. **戰鬥獲勝**
   - 擊敗等級相近對手 = +5-20 進度
   - 等級越高傷害越大（有助於突破）

3. **奇遇**（由 AI 設計）
   - 獲得秘籍、丹藥、靈石等 = +20-50 進度
   - 機率與「氣運值」正相關

4. **任務與劇情**
   - 完成宗門任務 = +10-30 進度
   - 關鍵劇情事件 = +50-100 進度

## 6. 氣運系統 (Karma)

氣運值影響遊戲的隨機性和奇遇頻率。

### 氣運來源

| 行為 | 影響 |
|------|------|
| 救助弱者 | +5 |
| 欺負弱小 | -10 |
| 與 NPC 好感互動 | +1 |
| 背棄承諾 | -20 |
| 完成困難任務 | +10 |
| 逃離戰鬥 | -3 |

### 奇遇觸發

```
奇遇機率 = 基礎 10% + 氣運 × 1%

範例：
- 氣運 0: 10% 機率
- 氣運 +20: 30% 機率
- 氣運 -30: 0% 機率（無奇遇）
```

### 可能的奇遇

- 發現秘藏寶藏
- 邂逅神秘人物
- 獲得失傳秘籍
- 古老妖獸相助
- 師傅突然教授絕招
- 身體自動進化

## 7. NPC 關係系統

### 親密度機制

```
初始親密度：-20 ~ +20（取決於 NPC 性格）

親密度變化：
- 每次成功互動：+1 ~ +10
- 每次衝突：-5 ~ -20
- 時間衰減：每 10 天 -1（代表 NPC 淡忘）
```

### 親密度影響

| 親密度 | 態度 | 效果 |
|--------|------|------|
| < -20 | 敵視 | 可能被攻擊、禁入地點 |
| -10 ~ 0 | 冷漠 | 拒絕交易、對話簡短 |
| 0 ~ +10 | 中立 | 正常交互 |
| +10 ~ +30 | 友善 | 交易折扣、提供幫助 |
| +30 ~ +50 | 親近 | 傳授技能、特殊任務 |
| +50+ | 摯友 | 可能成為伴侶、獲得終極秘法 |

## 8. 宗門設定 - 青雲門

### 宗門等級

Tier 5（最頂級宗門之一）

### 組織結構

```
掌門（玄靈子）Tier 5.5
    |
    ├─ 長老會（3 名）
    │   ├─ 首席長老（修為 Tier 4.8+）
    │   ├─ 執法長老
    │   └─ 內務長老
    │
    ├─ 內門弟子（100+ 人）
    │   └─ 包括天才弟子如紅藝
    │
    ├─ 外門弟子（500+ 人）
    │   └─ 包括新手玩家
    │
    ├─ 雜役弟子（1000+ 人）
    │   └─ 負責後勤
    │
    └─ 附屬組織
        ├─ 藥王谷（盟友）
        ├─ 異族勢力（如魔門臥底）
        └─ 商業合作者
```

### 宗門規則

1. **修為優先**：高修為享受特權
2. **宗門至上**：背叛宗門者必死
3. **實力為尊**：所有事務可決鬥解決
4. **禁忌**：
   - 不可傷害同門弟子（除非決鬥）
   - 不可偷盜宗門秘籍
   - 不可與魔門來往（但無人監管）

## 9. 潛伏的秘密

### 掌門玄靈子的真相

- 他的舊傷來自「昆侖秘戰」中的一次背叛
- 宗門內有魔門臥底
- 他正在秘密尋找「心悟仙草」治癒內傷
- 他知道下一代掌門候選人是誰（紅藝）

### 魔門的陰謀

- 派遣使者（無名）監視青雲門
- 計畫在宗門內部製造分裂
- 尋找「天選之人」進行黑暗儀式

### 隱士的秘密

- 石長眉是曾被陷害的前長老
- 他掌握著青雲門政治鬥爭的真相
- 他在研究「天外隕石」的力量（可能是天劫源頭）

## 10. Phase 1 限制與 Phase 2 方向

### Phase 1（當前）

- 單機遊戲，只有一個玩家
- 只能在固定的 10 個 NPC 和 9 個地點活動
- NPC 狀態靜態（不會隨著你的行動改變其他 NPC 的狀態）
- 故事線相對獨立

### Phase 2 計劃

- 引入「痕跡系統」（多個玩家在同一地點會互相影響）
- NPC 動態化（NPC 會根據全體玩家行為改變狀態）
- 派系系統（玩家可選擇支持不同派系）
- 宗門戰爭（玩家可參與宗門內部或對外戰爭）
- 連鎖任務（完成一個玩家的任務會影響其他玩家的遊戲）
